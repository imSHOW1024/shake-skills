# D4 few-shot 範例（二）：講評較完整 + 行政資訊較重

用途：當課堂同時出現「多組講評」與「較大量行政 / 行程 / 參訪資訊」時，示範如何維持資訊完整，但仍不把簡報本體重講一遍。

請模仿以下幾件事：
- 講評段落可以比第一份示範更完整，但仍只寫**真正影響理解與後續準備**的內容
- 每組仍固定收成「**核心建議** / **指正或補充**」兩層
- 若有行政 / 行程資訊很多，允許獨立拆成：`行程總覽 / 費用說明 / 注意事項 / 待辦 & 截止`
- 仍然遵守：儘量還原原意、不替講者補完因果、不自行延伸評論、不把簡報本體重講一遍

## 示例輸出（節錄）

### 老師講評摘要

#### 第一組：公司 A
- **核心建議：**
  - 先補公司背景，再談海外布局，讓聽眾先知道這家公司在做什麼。
  - 最後一頁要能回到人才管理或合作意義，呼應課程設計目的。
- **指正或補充：**
  - 地點、產能、規模等關鍵數字要寫清楚，不要只用模糊描述。
  - 若簡報用了數字，資料來源要補上。

#### 第二組：品牌 B
- **核心建議：**
  - 若影片和口頭簡報重複，就以口頭簡報為主，影片只保留最必要的品牌介紹。
  - 可以補全球 / 區域 / 當地三層級對照數字，讓聽眾知道市場差距。
- **指正或補充：**
  - 競爭對手要點名真正同級的品牌，不要只寫模糊的「其他品牌」。
  - 若某個數字很亮眼但聽起來不太合理，要先確認來源再用。

#### 第三組：集團 C
- **核心建議：**
  - 先把規模、營收、員工數、公開資料能看到的關鍵資訊講清楚。
  - 最後要把報告收斂到：聽眾接下來能怎麼討論、怎麼提問。
- **指正或補充：**
  - 具體圖像比示意圖更有說服力。
  - 若談通路、零售或海外版圖，可以點出哪些脈絡值得往東南亞延伸觀察，但不要把推論寫得太滿。

### 老師補充的跨文化觀點
- 先收斂成 2 到 4 個最值得記的框架或提醒，不要把整段補充講散。
- 若有工作坊 / 分組討論設計，要寫清楚討論主軸與雙方視角。

### 行程總覽
- 若課堂明確交代了日期、上午 / 下午安排、交流單位、參訪點，可直接用表格整理。

### 費用說明
- 只列真正影響決策或準備的費用資訊，例如誰負擔、哪些要自付、匯率或現金使用建議。

### 注意事項
- 依主題拆開，例如：護照與簽證 / 住宿 / 交通 / 購物與消費 / 實用資訊。
- 只寫之後真的會回頭看的提醒。

### 待辦 & 截止
- [ ] 有截止日的待辦要直接把時間寫進去。
- [ ] 無截止日但出發前一定要做的，也要列清楚。
