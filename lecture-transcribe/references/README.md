# lecture-transcribe references index

## Few-shot 範例索引

### D4（報告講評 + 行政/行程混搭）

1. `few-shot-d4-mixed.md`
   - 用途：一般型 D4 混搭示範
   - 偏好：講評 + 共同提醒 + 補充 + 行程/注意事項/待辦
   - 重點：儘量還原原意、不補完因果、不自行延伸評論、不重講簡報本體

2. `few-shot-d4-crossculture-lecture-heavy.md`
   - 用途：講評較完整、行政資訊較重的 D4 示範
   - 偏好：除了講評外，允許拆成 `行程總覽 / 費用說明 / 注意事項 / 待辦 & 截止`
   - 重點：資訊可更完整，但仍只保留真正影響理解與後續準備的內容

## 使用原則

- few-shot 用來示範**章節順序、資訊密度、措辭控制、還原程度**
- 可以模仿風格與結構，**不能照抄內容**
- 若使用者在訊息內另寫 `特記事項 / 摘要要求 / 補充要求`，那些規則優先於 few-shot

## 維護原則

- 若之後又出現穩定且高品質的新寫法，可以新增新的 few-shot 檔
- 新增後，記得同步更新：
  - `summary_prompts.py` 的 `_FEW_SHOT_FILES`
  - 本索引檔
  - `SETUP.md` 的 Few-shot 範例段落
