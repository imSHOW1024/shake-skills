# D4 few-shot 範例：報告講評 + 行政/行程混搭

用途：當逐字稿同時包含「多組報告講評」與「行政 / 行程 / 參訪 / 備案 / 注意事項」時，示範理想輸出長相。

請模仿以下幾件事：
- 先寫**講評摘要**，再寫**共同提醒**、**補充**、**行程 / 注意事項 / 待辦**
- 每組固定收成「**核心建議** / **指正或補充**」兩層
- 儘量還原原意，不替講者補完因果，不自行延伸評論
- 主語已經明確時，不要每句都重複寫「老師…」
- 不把簡報本體重講一遍，只保留理解講評所需的最低必要脈絡

## 示例輸出（節錄）

### 英文簡報老師講評摘要

#### 第一組：公司 A
- **核心建議：**
  - 簡報時間要抓緊，Q&A 至少預留 3 到 5 分鐘。
  - 可以直接做一頁 5 個討論問題，讓現場比較容易進入 discussion。
- **指正或補充：**
  - 如果站在學生角度，最可能會追問的是：為什麼選這個市場或這個投資地點。
  - 這題要先準備回答，不要等現場才想。

#### 第二組：品牌 B
- **核心建議：**
  - Q&A / 討論頁要直接放在投影片上，不要只有單向報告。
- **指正或補充：**
  - 若牽涉多國市場，進入模式差異要講清楚；有些是 intermediary，有些比較接近 direct investment。

#### 第三組：集團 C
- **核心建議：**
  - 可補產品圖片、主要客戶，讓聽眾更快理解公司輪廓。
- **指正或補充：**
  - 若談人才或培訓，最好補可查證的公開資訊，不要停在空泛描述。

### 三組共同提醒
- 不只要有問題，也要先準備幾個 general questions 的回答。
- 交流重點不是把簡報講完，而是要把對方帶進互動。

### 老師補充
- 這類交流本來就可能有變動，很多安排不一定會很早定案。
- 南方國家的節奏本來就比較有彈性，不要用完全照表操課的期待去看。

### 行程 / 行政 / 任務說明
#### 可能變動 / 備案
- 原行程若取消，要直接寫清楚備案是什麼、改到哪一天、和誰交流。

#### 注意事項
- 飲食、交通、時間節奏、在地溝通方式，只寫真正會影響現場狀態的提醒。

#### 待辦
- [ ] 補 Q&A / 討論問題
- [ ] 先準備現場最可能被問到的回答
